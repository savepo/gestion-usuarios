# Gestión de Usuarios

Aplicación web para gestión de usuarios con registro, login y edición de perfil.

## Instrucciones para Clonar y Ejecutar Localmente

1. **Clonar el Repositorio:**
   - Si está en GitHub, clona con: `git clone https://github.com/savepo/gestion-usuarios.git`
   - O descarga el ZIP y extrae en `C:\xampp\htdocs\`.

2. **Configurar XAMPP:**
   - Instala XAMPP si no lo tienes.
   - Inicia Apache y MySQL.

3. **Configurar Base de Datos:**
   - Ve a phpMyAdmin y crea la DB `gestion_usuarios`.
   - Ejecuta el SQL para la tabla users (ver documentacion_proyecto.md).

4. **Ejecutar:**
   - Ve a `http://localhost/gestion-usuarios/`.

5. **Credenciales por Defecto:**
   - Usuario: root
   - Contraseña: (vacía)

Asegúrate de que el proyecto esté en htdocs para que funcione localmente.
